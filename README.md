# MJX School System

A modern school management system built with **React**, **Vite**, **TypeScript**, **Tailwind CSS**, and **Material UI**.

MJX School System contains:
- Student Portal
- Instructor Portal
- Admin Portal

The project is focused on building a professional frontend-only school ERP/LMS system with modern UI/UX design.
