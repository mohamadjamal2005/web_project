# MJX School System

A modern school management system built with **React**, **Vite**, **TypeScript**, **Tailwind CSS**, and **Material UI**.

MJX School System contains:
- Student Portal
- Instructor Portal
- Admin Portal

The project is focused on building a professional frontend-only school ERP/LMS system with modern UI/UX design.

---

# 🚀 Tech Stack

- React
- Vite
- TypeScript
- Tailwind CSS
- Material UI (MUI)
- React Router DOM
- Recharts

# 🛠️ Installation

## 1️⃣ Clone the repository

```bash
git clone https://github.com/mohamadjamal2005/web_project
```

---

## 2️⃣ Navigate to the project

```bash
cd web_project
```

---

## 3️⃣ Install dependencies

```bash
npm install
```

---

## 4️⃣ Start development server

```bash
npm run dev
```

---

# 📦 Required Packages

```bash
npm install react-router-dom
npm install @mui/material @emotion/react @emotion/styled
npm install @mui/icons-material
npm install recharts
```
